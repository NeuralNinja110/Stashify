# STASHIFY

## A Multiplayer Cognitive Game Platform & Memory Companion for Elderly Care

### Overview

**Stashify** is a **non-medical, AI-assisted cognitive engagement platform** designed for senior citizens (60+).
The platform combines **memory games, AI voice companions, reminiscence journaling, and social multiplayer experiences** to help elderly users maintain cognitive activity, emotional well-being, and social connection.

Stashify is built on the philosophy:

> **“Gentle challenge. Emotional safety. Familiarity. Repetition. Social connection.”**

The system is designed specifically for seniors, including those with **low digital literacy**, ensuring accessibility through **voice-first interaction, large UI elements, and simple navigation**.

---

# Problem Statement

Many elderly individuals face:

* Memory decline
* Reduced social interaction
* Difficulty using modern apps
* Lack of emotionally engaging digital experiences
* Limited access to cognitive stimulation tools
* Early signs of Mild Cognitive Impairment (MCI) going unnoticed

Existing cognitive training apps often feel **clinical, complex, or impersonal**, resulting in **low engagement and inconsistent usage**.

**Stashify addresses this gap** by combining cognitive training with emotional engagement and social interaction.

---

# Core Features

## 1. AI Voice Companion — Thunaivan / Thunaivi

An AI-based conversational companion designed to:

* Engage users in daily conversations
* Encourage memory recall through personalized prompts
* Provide emotional encouragement
* Ask context-aware questions based on past interactions
* Speak in a friendly, supportive tone

The AI remembers:

* User interests
* Family members
* Past memories (Golden Moments)
* Game performance
* Important dates and reminders

---

## 2. Golden Moments — Voice Memory Journal

A voice-based reminiscence system where users can:

* Record life stories
* Upload photos
* Talk about memories
* Store personal experiences
* Replay memories anytime

The system processes:

* Speech-to-text transcription
* Emotion tagging
* Memory tagging (people, places, events)
* Follow-up AI conversations based on memories

This helps reinforce **episodic memory and emotional well-being**.

---

## 3. Cognitive Games Suite

Games are designed to train different cognitive functions:

| Game             | Cognitive Skill             |
| ---------------- | --------------------------- |
| Memory Grid      | Visual & Short-term Memory  |
| Word Chain       | Language & Processing Speed |
| Echo Chronicles  | Working Memory              |
| Riddles          | Reasoning                   |
| Family Tree Quiz | Long-term & Semantic Memory |
| Letter Link      | Language & Attention        |

The system includes **adaptive difficulty**, ensuring the games remain challenging but not frustrating.

---

## 4. Multiplayer & Social Engagement

To reduce social isolation, Stashify includes:

* Age-grouped leaderboards
* Multiplayer word and memory games
* Emoji-based interaction (no toxic chat)
* Voice encouragement between players
* Cooperative and competitive modes

---

## 5. Family Tree Memory System

Users can add:

* Family members
* Photos
* Relationships
* Important dates

The system then generates **memory-based quizzes** to help users remember family connections and events.

---

## 6. Reminder & Routine System

Users receive reminders for:

* Medicines
* Appointments
* Daily tasks
* Events

Voice confirmation ensures clarity and reassurance.

---

## 7. Cognitive Reports & Insights

The system generates **auto-generated cognitive reports** including:

* Game performance
* Memory analysis
* Attention & language metrics
* Trend graphs
* AI-generated insights
* Suggestions for improvement

Reports can be shared with **family members or caregivers**.
